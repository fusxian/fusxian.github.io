Java中的数组

Java基础知识系列课程-Java中的数组 @程序猿付显

  数组也是一种变量,它可以存储一组固定长度、相同数据类型的数据

一、一维数组

  • 声明
    • int[] array; –>声明类型为int的数组
    • int array[]; –>这种方式也是允许的
  • 初始化
    • array = new int[5]; –>初始化长度为5的数组
    • array = new int[]{0,1,2,3,4}; –>直接填入5个数据完成初始化和赋值操作
    • array = {0,1,2,3,4}; –>静态初始化,直接填入5个数据完成初始化和赋值操作
  • 赋值
    • array[0] = 0; –>使用下标从 [0] 开始进行赋值
  • 使用
    • int i = array[0]; –>此处也是使用下标从第一个[0]开始进行遍历的

数据在堆内存中的存储模型

二、二维数组

  二维数组就是一个特殊的一维数组，其每一个元素都是一个一维数组
  语法格式为:

  • int[] [] arr = new int[length1][length2];

    • length1 为行数
    • length2 为列数

以上代码执行完毕后再堆内存中的模型如图:

三、数组的使用

  以上数组演示的都是int类型的,我们还可以用其他任意的类型,包括但不限于八种基本数据类型、String、类等。

  • 数组是内存中有序数据的集合，数组中的每个元素具有相同的数据类型。
  • 数组必须先声明，对其初始化以后才能使用。对数组初始化有静态初始化和动态初始化两种方式。
  • 通过下标来访问数组中的每个元素。
  • 数组可以分为一维数组和多维数组。二维数组可以理解为一维数组的集合。
